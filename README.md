# The_OBJ_Project
SignBridge — двусторонний переводчик языка жестов и речи. Распознаёт жесты через камеру и озвучивает текст. Преобразует текст/голос в жесты через 3D-аватар. Поддержка разных жестовых языков, история переводов. Для доступной коммуникации между слышащими и людьми с нарушениями слуха.
