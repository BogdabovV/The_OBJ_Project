# The_OBJ_Project
SignBridge — двусторонний переводчик языка жестов и речи. Распознаёт жесты через камеру и озвучивает текст. Преобразует текст/голос в жесты через 3D-аватар. Поддержка разных жестовых языков, история переводов. Для доступной коммуникации между слышащими и людьми с нарушениями слуха.
План разработки проекта на 7 недель

Неделя 1: Инициация

· Настройка репозитория и среды
· Анализ существующих решений
· Определение core-функций
· Создание макетов интерфейса
· Выбор технологий

Неделя 2: Интерфейс

· Главное окно
· Переключение режимов
· Подключение камеры
· Отображение видеопотока

Неделя 3: Распознавание жестов

· Интеграция готовой модели (MediaPipe)
· Базовое распознавание простых жестов
· Вывод текста в интерфейс

Неделя 4: Звук

· Синтез речи
· Голосовой ввод
· Загрузка аудиофайлов

Неделя 5: Визуализация

· База жестов
· Отображение аватара/видео
· Воспроизведение последовательностей

Неделя 6: Сборка прототипа

· Объединение всех модулей
· Базовая история переводов
· Минимальные настройки

Неделя 7: ПРОТОТИП ГОТОВ

· Тестирование
· Исправление критических ошибок
· Демонстрационная версия
